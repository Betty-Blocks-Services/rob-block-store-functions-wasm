# Format endpoint result

TypeScript component that formats HTTP endpoint results. This component sends HTTP(S) requests with support for Liquid template rendering in URLs and request bodies, and formats the response.

## Prerequisites
* bun [install bun](https://bun.com/docs/installation)

## Building

Building can be done via:

```sh
. build.sh
```

## Running

There is an example script how to run the component:

```sh
. run.sh
```
